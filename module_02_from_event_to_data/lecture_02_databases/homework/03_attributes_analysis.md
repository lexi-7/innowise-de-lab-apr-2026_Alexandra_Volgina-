Шаг 2. Выбор и группировка атрибутов

Сущность products
product_id: числовой идентификатор, одно значение, не делимый
product_name: строковое название товара, не содержит повторяющихся групп или списков
price: число с плавающей точкой (или decimal), одно значение цены
category_id: целое число – внешний ключ к категории
class: один символ (A, B, C...), не список, не составное значение
modify_timestamp: дата и время последнего изменения, одно значение
resistant: булевый признак, одно значение
is_allergic: булевый признак, одно значение
vitality_days: целое число – срок годности в днях, одно значение

Сущность employees
employee_id: числовой идентификатор, одно значение
first_name: одно имя
middle_initial: одна буква, не список
last_name: одна фамилия
birth_date: одна дата рождения
gender: один символ (M/F)
city_id: один числовой ключ к городу
shop_id: один числовой ключ к магазину
hire_date: одна дата найма
