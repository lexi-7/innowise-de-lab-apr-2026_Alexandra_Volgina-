# Анализ атрибутов сущностей products и employees

## Сущность products

1. **product_id** - числовой идентификатор, одно значение, не делимый
2. **product_name** - строковое название товара, не содержит повторяющихся групп или списков
3. **price** - число с плавающей точкой (или decimal), одно значение цены
4. **category_id** - целое число, внешний ключ к категории
5. **class** - один символ (A, B, C...), не список, не составное значение
6. **modify_timestamp** - дата и время последнего изменения, одно значение
7. **resistant** - булевый признак, одно значение
8. **is_allergic** - булевый признак, одно значение
9. **vitality_days** - целое число, срок годности в днях, одно значение

## Сущность employees

1. **employee_id** - числовой идентификатор, одно значение
2. **first_name** - одно имя
3. **middle_initial** - одна буква, не список
4. **last_name** - одна фамилия
5. **birth_date** - одна дата рождения
6. **gender** - один символ (M/F)
7. **city_id** - один числовой ключ к городу
8. **shop_id** - один числовой ключ к магазину
9. **hire_date** - одна дата найма
