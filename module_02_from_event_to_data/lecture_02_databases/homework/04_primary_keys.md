# Первичные ключи (PK)

## Перечень первичных ключей

**countries** → country_id
- Уникальный идентификатор страны

**cities** → city_id
- Уникальный идентификатор города

**shops** → shop_id
- Уникальный идентификатор магазина

**employees** → employee_id
- Уникальный идентификатор сотрудника

**customers** → customer_id
- Уникальный идентификатор покупателя

**categories** → category_id
- Уникальный идентификатор категории

**products** → product_id
- Уникальный идентификатор продукта

**sales** → sales_id
- Уникальный идентификатор записи о продаже

## Обоснование (Customers)

Первичный ключ `customer_id` уникально идентифицирует каждого покупателя. Даже если у двух покупателей совпадают другие данные (имя, фамилия, адрес), их `customer_id` будет разным.

## Обоснование (Categories)

Первичный ключ `category_id` уникально идентифицирует каждую категорию товаров. Даже если названия категорий будут совпадать, ID останется уникальным.
